Requêtage d'API

  • DevAPI Attention, limitation à 100 requêtes par mois. Si le code ne fonctionne pas ouvrir un nouveau compte avec une nouvelle adresse mail pour générer une nouvelle clé.
  • AlphaVantage Attention limitation à 25 requêtes par jour. Si le cod ne fonctionne pas... retenter demain.
  • Gestion de la sécurité des clés API gérées comme variable d'environnement en les placant dans le dossier .env/ lui même placé dans le git ignore. Puis utilisation du package dotenv.
  • Récupération de json pour les news et les quotations.
  • Ajout d'une fonctionnalité de recherche du code ticker ou symbol à partir du mot saisi par l'utilisateur (à affiner éventuellement en ajoutant un sous niveau proposant tous les choix trouvés). Pour le moment, la fonction renvoie la première valeur.

Mise en place d'un outil de classification d'images (digits MNIST)

  • Bases d'images de digit MNIST keras (70Kimages)
  • Création d'un modèle CNN et entrainement
  • Taux d'accuracy 98%
  • Réalisation d'un programme de génération d'images digit selon plusieurs polics manuscrites choisies sur google fonts et téléchargées en ttf. Fond Noir, dessin blanc.
  • Test du modèle sur ces images par ajout d'un formulaire
  • Elargissement des fonctionnalités en créant un canva permettant de dessiner un digit et de le faire deviner par l'algorithme
